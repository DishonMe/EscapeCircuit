\section*{3-Bit Synchronous Counter}
Implement a 3-bit synchronous counter using D-Flip-Flops (\textbf{DFF}).

\textbf{Requirements:}
\begin{itemize}
\item The counter has one input, I, which acts as an enable signal.
\item When I=1, the counter increments its value on each clock cycle.
\item When I=0, the counter holds its current value.
\item The counter wraps around from 7 (binary 111) back to 0 (binary 000).
\end{itemize}

\textbf{Outputs:}
Output_0, Output_1, Output_2 represent the 3-bit binary value, with Output_2 being the Most Significant Bit (MSB).

\textit{Hint: Think about what condition causes each specific bit to flip (toggle) its state.}
